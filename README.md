# Expense Tracker (MERN Stack)

**Expense Tracker** is a full-stack web application built with **MongoDB, Express, React, and Node.js (MERN)**.  
It allows users to track their **expenses and income**, categorize them, and visualize data through multiple responsive charts.  

---

## **Features**

- **User Authentication**: Signup and login with JWT token.
- **Expense Management**:
  - Add, edit, and delete expenses.
  - Assign expenses to categories.
  - Track income and expenses separately.
- **Category Management**:
  - Add, edit, and delete categories.
  - Assign expenses to categories.
- **Analytics & Charts**:
  - **Pie Chart**: Expenses by category.
  - **Bar Chart**: Income vs Expense per category.
  - **Line Chart**: Expenses & Income over time.
  - **Doughnut Chart**: Expense vs Income ratio.
- **Responsive Design**:
  - Works on desktop, tablet, and mobile devices.
- **Local MongoDB Integration**:
  - Auto-created database and collections.
- **Secure API**:
  - All routes protected with JWT authentication.
- **Error Handling**:
  - Safe handling for missing categories, token validation, and empty datasets.

---

## **Tech Stack**

- **Frontend**: React, Axios, react-chartjs-2, Chart.js  
- **Backend**: Node.js, Express  
- **Database**: MongoDB (local)  
- **Authentication**: JWT, bcrypt  
- **Styling**: CSS, responsive layout
